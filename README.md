# 📊 Sales & Order Analysis Dashboard (Excel)

---

### 📌 Overview

This project is an interactive Excel dashboard built to analyze e-commerce sales and order data (1.28L+ records).
The goal is to track performance, identify trends, and support better business decisions.

### 🗂️ Dataset

The dataset contains detailed order-level information, including:

* Order Details: Order ID, Date, Status, Order Status
* Product Info: Category, Style, SKU, Size, ASIN
* Sales Metrics: Quantity, Amount, Currency
* Logistics: Courier Status, Fulfilment, Ship Service Level
* Customer Location: City, State, Postal Code, Country
* Channel Info: Sales Channel, B2B, Promotion IDs

  ----

### ⚙️ Data Preparation

* Cleaned and transformed using Excel Power Query
* Handled missing values & inconsistencies
* Standardized formats (dates, text, categories)
* Created derived columns for better analysis
* Validated data for accuracy and consistency

----
### 📊 Dashboard Features

The dashboard provides a clear and interactive view of:

* Sales Performance
* Total Sales
* Total Quantity
* Total Customers
* Order Insights
* Shipped vs Cancelled orders
* Order status breakdown
* Category Analysis
* Sales distribution across product categories
* Geographical Insights
* State-wise sales distribution (India map)
* City-level contribution
* Fulfilment Analysis
* Amazon vs Merchant fulfilment comparison

  ---
  
### 🎛️ Filters & Interactivity
* State filter
* Category filter
* Order Status filter
* Fulfilment type filter

These allow users to drill down into specific segments easily.

### 🛠️ Tools & Skills Used

* Microsoft Excel

### 🎯 Key Insights

* Amazon fulfilment dominates overall order volume
* Certain states (like Telangana in view) show high sales concentration
* Kurtas and sets contribute majorly to sales
* Majority orders are successfully shipped vs cancelled
* Sales patterns vary significantly by region and category

This dashboard helps in:

* Monitoring business performance in real-time
* Identifying top-performing products & regions
